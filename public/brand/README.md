# Vertex Kashmir Holidays — Logo Kit

Vector-rebuilt from the original artwork. All wordmarks are outlined to paths
(no font files required to render). Mark geometry and palette are faithful to
the source logo.

## Palette
| Token        | Hex      | Use                                   |
|--------------|----------|---------------------------------------|
| Brand Navy   | #0B1F3A  | "Vertex" wordmark, dark surfaces, theme |
| Brand Blue   | #0A4DA2  | Mark – left facet                     |
| Brand Green  | #0BA45B  | Mark – right facet, tagline, accents  |
| Green (dark) | #16B364  | Tagline on dark backgrounds           |

## Typography
- Display / wordmark: **Playfair Display** SemiBold
- Tagline & UI:       **Inter** SemiBold (tracked caps for tagline)

## Folders
- `svg/`        — master vectors (horizontal, stacked, wordmark, icon; light/dark/mono)
- `png/`        — transparent PNG exports at 1×/2×/4×
- `favicon/`    — favicon.ico (16-64), favicon.svg, 16/32/48 PNG
- `app-icons/`  — PWA (192/512 + maskable), apple-touch-icon (180), mstile
- `social/`     — Open Graph 1200×630, square 1080, banner 1500×500, avatars 1024
- `web/`        — site.webmanifest, browserconfig.xml, head snippet, Next.js notes

## Logo usage rules
- Logo 1 (icon mark) → favicon, PWA icons, loading screen, admin
- Light surfaces → horizontal/stacked **light** (navy wordmark)
- Dark surfaces  → horizontal/stacked **dark** (white wordmark)
- Clear space ≥ height of the icon mark on all sides
- Minimum size: 120px wide (horizontal lockup), 28px (icon)
- Never recolour the mark, stretch, rotate, or add effects.
